The code contains 7 .py file. 

1. dt_small.py : Decision Tree Classifier trained on small dataset.

2. lr_small.py : Logistic Regression trained on small dataset.

3. nb_small.py : Naive Bayes trained on small dataset.

4. lr_final.py : Logistic Regression trained on full dataset.

5. visualization.py : Visualizations

6. pub.py : Publisher (Kafka)

7. sub.py : Subsriber (Kafka)


The file has one pdf, "Final_report" consisting of overview of the project and the results we obtained.
